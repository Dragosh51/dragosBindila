Gazetter

Country and information search web application that allows users to find details about various countries, including demographics, geographical data, and more.

The website has the following features:

  • Automatically selects the country that you are accessing the website from.
  • Displays the border of the country and marks the cities and airports of that country.
  • The website has 5 buttons:
    • Information: Shows the user general information about the country.
    • Weather: Shows the user the current weather and also a forecast.
    • News: Shows the user recent news from the selected country.
    • Wikipedia: Shows the user Wikipedia pages from the selected country.
    • Currency Calculator: Allows the user to convert dollars into the currency of the selected country.

Languages: HTML, CSS, JavaScript, PHP

Libraries: Leaflet, Bootstrap, JQuery

Tools: Git, GitHub

Company Directory

Company staff management web application that helps businesses keep track of employees, departments, and their roles within the organization.

The website has the following features:

  • A search bar that filters the currently displayed table dynamically.
  • A refresh button that refreshes the table.
  • A filter button that filters employees either by department or location. The button is disabled for the "Departments" and "Locations" nav tabs.
  • An add button that adds either an employee, a department, or a location depending on which nav tab is currently open.
  • All tables are populated from a SQL database and are ordered in alphabetical order.
  • All the entries in each table have an "Edit" and "Delete" button, allowing the user to update or delete an entry.
  • The "Delete" modal in the "Departments" and "Locations" nav tabs will not allow the user to delete an entry if the entry has any dependencies.

Languages: HTML, CSS, JavaScript, PHP

Database: SQL

Libraries: Bootstrap, JQuery

Tools: Git, GitHub
